# Suggested Commands

## Package Management (uv)
- `uv init` тАФ Initialize a new project
- `uv add <package>` тАФ Add a dependency
- `uv sync` тАФ Sync/install all dependencies
- `uv run main.py` тАФ Run the main entry point

## Running the Agent
- `uv run main.py` тАФ Start the ADK Dev UI server

## Development
- `uv run python -c "..."` тАФ Quick Python one-liners
- `uv run pytest` тАФ Run tests (if configured)

## System Utils (macOS/Darwin)
- `git`, `ls`, `cd`, `grep`, `find` тАФ Standard unix tools
- `pbcopy`, `pbpaste` тАФ Clipboard (macOS)
